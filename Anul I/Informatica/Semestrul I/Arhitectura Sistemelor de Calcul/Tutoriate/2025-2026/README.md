# Arhitectura Sistemelor de Calcul
Pagina tutoriatului *susținut de Matei-Iulian Cocu și Chiper Ștefan* la Arhitectura Sistemelor de Calcul @ FMI 2025-2026.

Cum salvați mai ușor toate fișierele din repository *(din orice terminal)*:
```
cd path/to/local_dir/
git clone https://github.com/mateipiratul/tutoriat_asc_2526
cd tutoriat_asc_2526
ls
```

| Fișier | Descriere |
|--------|-----------|
| `asm_book` | Professional Assembly Language, R. Blum |
| `ph_book` | Computer Organization and Design, 3rd. Ed, David A. Patterson, John L. Hennessy |
| `cheat_sheet_lab_asc` | Notițe, pe scurt, pentru testul de laborator (colocviu) |
| `ex_test_lab` | Exerciții pentru testul de laborator parțial |
| `seminare_asc` | Rezolvările seminarelor |
| `sinteza_lab_asc` | Rezumat cuprinzător al materiei de laborator |
| `cheat_sheet_curs_asc` | Notițe, pe scurt, pentru examenul scris |
| `curs_asc` | Materia de curs integrală |
