/*Folositi functia qsort pentru a ordona crescator prima jumÄƒtate a unui tablou de numere intregi È™i descrescÄƒtor a doua jumÄƒtatea aceluiaÈ™i tablou. */
#include <stdbool.h>
#include <stdlib.h>
#include <math.h>


int compcresc(const void *a, const void *b) {
return (*(int*)a - *(int*)b);
}

int compdesc(const void *a, const void *b) {
return (*(int*)b - *(int*)a);
}

// int compdesc(const int * a, const int * b)
// {
//     return *b - *a;
// }multe warning-uri!

int main() {
    int v[101];
    int n;
    scanf("%d",&n);
    for(int i=0;i<n;i++) {
        scanf("%d",&v[i]);
    }
    int m=n/2;
    
    qsort(v,m,sizeof(int),compcresc);
    
    qsort(v+m,n-m,sizeof(int),compdesc);
    
    for(int i=0;i<n;i++) {
        printf("%d ",v[i]);
    }

    return 0;
}

#include <stdio.h>
#include <stdlib.h>
#include <string.h>

int alfabetic(const void* x, const void* y)
{
    //return strcmp((char*)x, (char*)y);
    return *(char*)x-*(char*)y;
}

int main()
{
    char sir[200];
    printf("introduceti sirul: ");
    scanf("%s", sir);
    qsort(sir, strlen(sir), sizeof(char), alfabetic);
    printf("sirul sortat este: ");
    printf("%s", sir);

    return 0;
}
